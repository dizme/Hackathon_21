Use case description

Un ente pubblico può rilasciare una credenziale di avvenuta vaccinazione ad un cittadino, che potrà esibirla a chiunque tramite il suo DizMe wallet.

Per maggiori info consultare: Documentazione.


How we built it

Abbiamo creato un sistema che permette interagire con la blockchain di Algorand, attraverso la quale l'autorità competente potrà rilasciare un certificato di avvenuta vaccinazione sotto forma di NFT. Il sistema potrà poi inviare una credenziale al DizMe wallet.
