Use case description

Accesso rapido ad un'area riservata per un operaio specializzato.

How we built it

Tramite l'impiego del Dizme Wallet un operaio specializzato potrà dimostrare le sue credenziali di identità (Nome, cognome, data di nascita), qualifica professionale (saldatore) e appartenenza ad una azienda al passaggio ad un tornello che svolgerà il ruolo di Verifier di queste VC emesse dall'azienda
