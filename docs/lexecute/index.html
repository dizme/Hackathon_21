Use case description

Lexecute is a solution for smart legal contract lifecycle management and execution, which allows the online and automated management of a contractual relationship over its entire life cycle.

In a B2B configuration, it provides organizations that stipulate consumer contracts with a platform that can be used by end users (consumer customers of B2B organizations). The reference B2B markets are those that manage massive contracts: InsureTech, Fintech, Leasing, Credit, E-government etc.

Lexecute’s technological infrastructure allows to create an ecosystem of ancillary service providers (called “Oracles” or “Third-Party Providers”), which substantially increases the added value of the solution.

The B2B solution implemented addresses some critical points of contract management:

  1. onboarding of contract parties using Verifiable Credentials to get personal information and documents
  2. digital signature of contracts

How we built it

The integration between Lexecute and Dizme is realized by adding 2 new software components to the Lexecute architecture:

  1. a web application to guide users through the onboarding process
  2. a microservice that provides the access interface to the Dizme ecosystem functionalities

By using Dizme, Lexecute can now quickly onboard new users with their L1 verifiable credentials, issue new member credentials, and digitally sign contracts.
